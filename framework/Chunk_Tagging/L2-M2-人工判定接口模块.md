# 人工判定接口模块:ManualReviewInterfaceModule

@framework

## 1. 能力声明（Capability Statement）

- `C1` 文档ID输入能力：支持输入单个需要进行人工审核的文档ID。
- `C2` 数据库数据获取能力：支持通过数据库接口读取该文档下待人工审核记录及其对应文本块内容。
- `C3` 人工审核数据导出能力：支持将待人工审核记录及其对应文本块内容整理为人工审核数据并输出给外部模块。
- `C4` 人工确认结果输入能力：支持接收外部返回的人工确认结果。
- `C5` 审核结果写入能力：支持将人工确认后的审核结果写入数据库。
- `N1` 不提供前端交互能力：不负责登录、页面展示、页面跳转或人工审核界面的实现。
- `N2` 不提供自动评分能力：不负责重新计算文本块与标签之间的匹配评分。

## 2. 边界定义（Boundary / Parameter 参数）

- `P1` 文档ID边界：输入文档ID必须为单个合法文档标识。来源：`C1`。
- `P2` 文档ID数量边界：单次输入的文档ID数量为 `1`。来源：`C1`。
- `P3` 数据库接口边界：输入数据库接口必须满足 `L1.M1[R4,R5]` 的组合要求。来源：`C2 + C5`。
- `P4` 待审核记录范围边界：读取出的记录必须限定为该文档下 `enable_status = 待人工审核` 且 `review_status = 未审核` 的记录。来源：`C2 + C3`。
- `P5` 审核数据字段边界：导出的人工审核数据必须至少包含 `doc_id`、`chunk_id`、`tag_name`、`confidence`、`reason`、`enable_status`、`review_status` 与对应文本块内容。来源：`C2 + C3`。
- `P6` 人工确认结果字段边界：输入的人工确认结果必须至少包含记录定位字段、人工最终结论与审核来源字段。来源：`C4`。
- `P7` 人工确认结果范围边界：人工最终结论只能为 `启用` 或 `不启用`，不得继续写回 `待人工审核`。来源：`C4`。
- `P8` 写入结果边界：写回数据库后的记录必须更新 `enable_status`、`review_status` 和 `review_source` 字段，其中 `review_status = 已审核`。来源：`C5`。

## 3. 最小结构基（Minimal Structural Bases）

- `B1` 文档ID：单个待人工审核文档的唯一标识。来源：`C1 + P1 + P2`。
- `B2` 数据库接口：读取该文档下待人工审核记录、关联查询对应文本块内容并写回人工审核结果的数据库接口 L1.M1[R4,R5]。来源：`C2 + C5 + P3 + P4 + P5 + P8`。

## 4. 基组合原则（Base Combination Principles）

- `R1` 人工审核数据导出组合
  - `R1.1` 参与基：`B1 + B2`。
  - `R1.2` 组合方式：接收单个文档ID后，通过数据库接口读取该文档下全部待人工审核记录，再关联查询这些记录对应的文本块内容，并整理为统一人工审核数据导出给外部模块。
  - `R1.3` 输出能力：`C1 + C2 + C3`。
  - `R1.4` 参数绑定：`P1 + P2 + P3 + P4 + P5`。

- `R2` 人工确认结果回写组合
  - `R2.1` 参与基：`B1 + B2`。
  - `R2.2` 组合方式：接收外部返回的人工确认结果，校验结果字段与人工最终结论是否合法，再通过数据库接口将对应记录写回数据库。
  - `R2.3` 输出能力：`C1 + C4 + C5`。
  - `R2.4` 参数绑定：`P1 + P2 + P3 + P6 + P7 + P8`。

## 5. 验证（Verification）

- `V1` 推导一致性：`B1` 必须可由 `C1 + P1 + P2` 推导得到，`B2` 必须可由 `C2 + C5 + P3 + P4 + P5 + P8` 推导得到。
- `V2` 规则一致性：每个 `R*` 必须明确参与基/组合方式/输出能力或失效结论/参数绑定。
- `V3` 结构闭环性：`R1 + R2` 的输出能力并集必须覆盖 `C1~C5`。
- `V4` 查询结果边界符合性：任意有效人工审核数据导出都必须满足 `P4 + P5`。
- `V5` 回写结果边界符合性：任意有效人工确认结果写回都必须满足 `P6 + P7 + P8`。
- `V6` 结构纯度：本模块只负责人工审核段，不得混入自动评分、自动判定或完整流程编排语义。
- `V7` 最小必要性：移除 `B1` 或 `B2` 任一结构基后，人工审核数据导出组合或人工确认结果回写组合至少一项无法成立。
- `V8` 结论表达：逐条输出 `R* -> C* + 参数绑定` 或 `R* -> N* + 参数绑定` 的通过或失败结论。
