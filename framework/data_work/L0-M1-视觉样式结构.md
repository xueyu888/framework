# 视觉样式结构:VisualStyle

@framework

## 0. 目标 (Goal)

- 视觉样式结构负责让一个已经成立的界面单元在页面中具备默认样式。
- 该结构聚焦“谁的样式 -> 什么样式”的结构成立过程。不是承担布局样式，状态流转样式，交互反馈样式职责。

## 1. 最小结构基（Minimal Structural Bases）

- `B1` 样式归属：界面单元的样式需要归属于明确单元，使样式不悬空。
- `B2` 外观样式：界面单元的边框、背景、阴影和文字样式。

## 2. 基排列组合（Base Arrangement / Combination）

- `R1` `默认样式成立`：由 `{B1, B2}` 形成界面单元完整的默认视觉样式。
