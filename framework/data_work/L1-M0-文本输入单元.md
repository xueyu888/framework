# 文本输入单元:TextInputUnit

@framework

## 0. 目标 (Goal)

- 文本输入单元负责让一个界面单元在页面中作为可输入文本的单元成立。

## 1. 最小结构基（Minimal Structural Bases）

- `B1` 界面单元：`L0.M0[R3]`
- `B2` 视觉样式：`L0.M1[R1]`
- `B3` 输入支持：`L0.M2[R2]`

## 2. 基排列组合（Base Arrangement / Combination）

- `R1` `文本输入单元成立`：由 `{B1, B2, B3}` 形成一个在页面中成立、具备默认样式并能够接收文本输入的文本输入单元。
