# 布尔表达式计算器: BooleanExpressionCalculator

@framework

## 0. 目标 (Goal)

接收一个布尔表达式字符串和 N 个等长布尔序列，按表达式定义的组合方式，逐位置运算，输出一个等长布尔结果序列。


## 1. 最小可行基（Minimum Viable Bases）

- `B1` 布尔表达式树：带归属关系的表达式结构
- `B2` 树构造器：根据符号序列构建布尔表达式树实例
- `B3` 求值器：对布尔表达式树进行递归计算，输出结果序列
