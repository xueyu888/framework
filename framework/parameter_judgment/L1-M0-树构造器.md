# 树构造器:TreeConstructor

@framework

## 0. 目标 (Goal)

接收布尔表达式字符串，输出一个布尔表达式树实例


## 1. 最小可行基（Minimum Viable Bases）

- `B1` 归属解析器：将布尔表达式字符串转换为符号序列
- `B2` 树构造器：根据符号序列构建布尔表达式树实例


## 2. 基组合原则（Base Combination Principles）

- `R1` B1 的输出是 B2 的输入
- `R2` B2 的输出即为最终输出


## 3. 边界定义（Boundary / 参数）

### 3.1 接口定义（IO / Ports）

- 输入：布尔表达式字符串
- 输出：一个布尔表达式树实例

### 3.2 参数边界（Parameter Constraints）

- `P1` 合法的布尔表达式字符串定义：
  - 操作数：一个或多个连续字母
  - 运算符：&&（与）、||（或）、!（非）
  - 分组符：(、)
  - 分隔符：空格
  - 优先级：! > && > ||，左结合
