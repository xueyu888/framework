# 树构造器:TreeConstructor

@framework

## 0. 目标 (Goal)

接收符号序列，输出一个布尔表达式树（即归属关系）


## 1. 最小可行基（Minimum Viable Bases）

- `B1` 映射表：一个输入对应一个确定的输出
- `B2` 比较器：用于比较两个元素的映射值大小
- `B3` 栈：后进先出的数据结构


## 2. 基组合原则（Base Combination Principles）

- `R1` 逐个遍历输入序列，组合使用 `B1`、`B2`、`B3` 进行元素比较和归属关系确定。


## 3. 边界定义（Boundary / 参数）

### 3.1 接口定义（IO / Ports）

- `输入`：符号序列
- `输出`：归属关系序列（布尔表达式树）

### 3.2 参数边界（Parameter Constraints）

- `P1` 映射表配置：
  - 输入域：运算符（&&、||、!）
  - 输出域：自然数
  - 映射规则：
    - ! → 3
    - && → 2
    - || → 1

- `P2` 符号序列长度：uint64
