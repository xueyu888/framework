# 元素标记器:ElementMarker

@framework

## 0. 目标 (Goal)

从布尔表达式字符串中提取出所有有意义的元素，并标记它们的身份（运算符、操作数、分组符）


## 1. 最小可行基（Minimum Viable Bases）

- `B1` 映射表：一个输入对应一个确定的输出
- `B2` 集合：持有一组确定的元素，对于任意给定的元素，能判断它在不在其中


## 2. 基组合原则（Base Combination Principles）
- `R1` 通过 `B1` 对输入序列逐元素映射，连续相同输出的元素累积成片段，输出不同则切断，产出片段序列
- `R2` 每个片段通过 `B2` 判断是否为其成员，以此确定片段的身份标记
- `R3` `R1` 的产出是 `R2` 的输入


## 3. 边界定义（Boundary / 参数）

### 3.1 接口定义（IO / Ports）
- `输入`：布尔表达式字符串
- `输出`：符号序列，每个符号携带身份标记（运算符 / 操作数 / 分组符）

### 3.2 参数边界（Parameter Constraints）

- `P1` 布尔表达式映射表配置：
  - 输入域：a-z、A-Z、&、|、!、(、)、空格
  - 输出域：字母、符号、分组符、空白
  - 映射规则：
    - a-z、A-Z → 字母
    - &、|、! → 符号
    - (、) → 分组符
    - 空格 → 空白

- `P2` 布尔运算符集合配置：
  - 成员：&&、||、!

- `P3` 输入字符串长度：uint64
