# 布尔表达式树:BoolExpressionTree

@framework

## 0. 目标 (Goal)

定义一个布尔表达式的结构


## 1. 最小可行基（Minimum Viable Bases）

- `B1` 二叉树：每个节点最多有两个子节点的树


## 2. 基组合原则（Base Combination Principles）

- `R1` 有且仅有一个B1


## 3. 边界定义（Boundary / 参数）

### 3.1 接口定义（IO / Ports）

### 3.2 参数边界（Parameter Constraints）

- `P1` 二叉树节点功能配置：
  - 叶子节点持有操作数
  - 非叶子节点持有一个布尔运算符（与/或/非）
    - 非叶子节点持有与时，有且仅有两个子节点
    - 非叶子节点持有或时，有且仅有两个子节点
    - 非叶子节点持有非时，有且仅有一个子节点
