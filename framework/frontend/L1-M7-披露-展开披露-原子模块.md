# 前端展开披露原子模块:FrontendDisclosureExpansionAtoms

@framework

模块短标签：展开披露
典型落地对象：accordion / disclosure / collapsible section
不包括：互斥视图切换、临时展开选择面、页面级导航

本模块定义前端展开披露原子。它承接手风琴、局部披露块和可折叠说明区这类“围绕局部标题或摘要进行展开与折叠”的结构，不把互斥视图切换、页面级导航或长期驻留多区布局混进同一模块。

## 1. 能力声明（Capability Statement）

- `C1` 披露入口承接能力：把标题行、摘要区与展开触发入口收敛为稳定披露入口结构。
- `C2` 展开闭合承接能力：把内容区附着、折叠/展开状态与切换成立条件收敛为稳定局部展开闭合结构。
- `C3` 展开可达能力：把键盘切换、当前展开提示与展开后阅读路径收敛为稳定可达结构。

### 非职责声明（Non-Responsibility Statement）

- `N1` 非职责声明：不负责互斥视图配对、页面级跳转与临时任务接管流程。

## 2. 边界定义（Boundary / 参数）

- `DISCLOSURE` 披露边界：标题行、摘要位、展开入口与入口顺序需稳定。来源：`C1 + C3`。
- `CONTENT` 内容边界：展开内容区、折叠后内容关系、内容摘要与内容归属关系需稳定。来源：`C1 + C2`。
- `STATE` 状态边界：折叠、展开与展开中等状态集合、状态迁移关系与反馈窗口需稳定。来源：`C2 + C3`。
- `TOGGLE` 切换边界：展开出口、折叠出口、再次切换出口与切换成立条件需稳定。来源：`C1 + C2`。
- `FOCUS` 焦点边界：标题聚焦、展开后阅读路径、回退落点与焦点回路需稳定。来源：`C2 + C3`。
- `A11Y` 可访问边界：当前展开提示、内容关系、动作说明与辅助技术可感知关系需可达。来源：`C1 + C3`。

## 3. 最小可行基（Minimum Viable Bases）

- `B1` 披露入口基：L0.M0[R2] + L0.M1[R4] + L0.M2[R1,R2]。来源：`C1 + DISCLOSURE + TOGGLE + STATE`。
- `B2` 展开闭合基：L0.M0[R1,R4] + L0.M1[R4] + L0.M2[R1,R3]。来源：`C2 + C3 + CONTENT + FOCUS + A11Y`。

## 4. 基组合原则（Base Combination Principles）

- `R1` 披露入口先行
  - `R1.1` 参与基：`B1`。
  - `R1.2` 组合方式：先固定标题行、摘要位与展开触发入口，形成稳定披露入口，再允许局部展开内容附着其上。
  - `R1.3` 输出能力：`C1`。
  - `R1.4` 边界绑定：`DISCLOSURE + TOGGLE + STATE`。
- `R2` 展开闭合成立
  - `R2.1` 参与基：`B1 + B2`。
  - `R2.2` 组合方式：展开内容必须围绕稳定披露入口附着，并通过折叠/展开状态、切换成立条件与阅读路径形成统一局部披露闭合结构。
  - `R2.3` 输出能力：`C1 + C2 + C3`。
  - `R2.4` 边界绑定：`CONTENT + STATE + FOCUS + A11Y + DISCLOSURE + TOGGLE`。
- `R3` 禁止组合
  - `R3.1` 参与基：`B1 + B2`。
  - `R3.2` 组合方式：只有标题行没有内容区、展开内容脱离披露入口存在、展开状态不可解释，或把互斥视图切换、临时展开选择面、页面级导航误写为展开披露原子的方案均无效。
  - `R3.3` 失效结论：`N1`。
  - `R3.4` 边界绑定：`DISCLOSURE + CONTENT + STATE + TOGGLE + FOCUS + A11Y`。

## 5. 验证（Verification）

- `V1` 披露入口完整性：披露入口结构必须满足 `DISCLOSURE + TOGGLE` 约束；标题行、摘要位与展开触发入口完整。
- `V2` 展开闭合有效性：局部展开结构必须满足 `CONTENT + STATE` 约束；展开内容区、折叠关系与当前状态可解释，且切换成立条件完整。
- `V3` 可达完整性：当前展开提示、阅读路径与回退落点必须满足 `FOCUS + A11Y` 约束；并对辅助技术可感知。
- `V4` 最小必要性：移除任一 `B*` 后，`V1 ~ V3` 至少一项失败。
- `V5` 结论表达：逐条输出 `R* -> C* + 边界绑定` 或 `R* -> N* + 边界绑定` 的通过或失败结论，并附对应边界证据。
