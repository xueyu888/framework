# 前端路径链导航原子模块:FrontendPathChainNavigationAtoms

@framework

模块短标签：路径回溯
典型落地对象：breadcrumb / path chain / hierarchical backtrack
不包括：一般集合浏览、页序列导航、互斥视图切换

本模块定义前端路径链导航原子。它承接面包屑、层级路径链和当前位置回溯链这类“围绕当前位置的路径链压缩表达与向上回溯关系成立”的结构，不把普通列表导航、页序列导航或页面级路由编排混进同一模块。

## 1. 能力声明（Capability Statement）

- `C1` 路径链承载能力：把路径片段、当前位置与层级链顺序收敛为稳定路径链承载结构。
- `C2` 回溯跳转闭合能力：把上游片段跳转、当前位置停驻、回溯出口与回溯后当前位置落位收敛为稳定回溯跳转闭合结构。
- `C3` 路径链可达能力：把路径朗读顺序、当前位置提示与跳转说明收敛为稳定可达结构。

### 非职责声明（Non-Responsibility Statement）

- `N1` 非职责声明：不负责页面级路由总编排、普通集合浏览树与页序列窗口计算。

## 2. 边界定义（Boundary / 参数）

- `CHAIN` 路径链边界：路径片段顺序、层级链长度、当前位置与路径片段到层级链的映射关系需稳定。来源：`C1 + C3`。
- `SEGMENT` 片段边界：路径片段标签、片段摘要、片段分隔关系与片段归属关系需稳定。来源：`C1 + C2`。
- `CURRENT` 当前位边界：当前位置停驻、当前位置摘要、不可跳转位与当前位判定条件需稳定。来源：`C1 + C3`。
- `RETURN` 回溯边界：上游跳转、层级返回、再次进入落点与回溯结果类型需稳定。来源：`C2 + C3`。
- `FOCUS` 焦点边界：路径片段聚焦、当前位置聚焦、回退落点与焦点回路需稳定。来源：`C2 + C3`。
- `A11Y` 可访问边界：路径朗读顺序、当前位置提示、跳转说明与辅助技术可感知关系需可达。来源：`C1 + C3`。

## 3. 最小可行基（Minimum Viable Bases）

- `B1` 路径链承载基：L0.M0[R1,R3] + L0.M2[R1,R2]。来源：`C1 + CHAIN + SEGMENT + CURRENT`。
- `B2` 回溯跳转闭合基：L0.M0[R4] + L0.M1[R4] + L0.M2[R1,R3]。来源：`C2 + C3 + RETURN + FOCUS + A11Y`。

## 4. 基组合原则（Base Combination Principles）

- `R1` 路径链先行
  - `R1.1` 参与基：`B1`。
  - `R1.2` 组合方式：先固定路径片段顺序、层级链关系、当前位置停驻位与路径片段到层级链的映射关系，再允许回溯跳转出口接入。
  - `R1.3` 输出能力：`C1`。
  - `R1.4` 边界绑定：`CHAIN + SEGMENT + CURRENT`。
- `R2` 回溯跳转闭合成立
  - `R2.1` 参与基：`B1 + B2`。
  - `R2.2` 组合方式：当前位必须能在路径链中被稳定判定，并通过上游片段跳转、回溯出口、回溯结果类型、回溯后当前位置落位与当前位提示形成统一路径链回溯结构；若不能沿既有路径链形成回溯关系，或退化为一般集合浏览、页序列导航或页面级路由编排，则该结构不成立。
  - `R2.3` 输出能力：`C1 + C2 + C3`。
  - `R2.4` 边界绑定：`RETURN + FOCUS + A11Y + CHAIN + SEGMENT + CURRENT`。
- `R3` 禁止组合
  - `R3.1` 参与基：`B1 + B2`。
  - `R3.2` 组合方式：没有当前位置却声明路径链、上游片段与当前位置不可区分、跳转说明缺位，或把一般集合浏览、页序列导航、互斥视图切换误写为路径链导航原子的方案均无效。
  - `R3.3` 失效结论：`N1`。
  - `R3.4` 边界绑定：`CHAIN + SEGMENT + CURRENT + RETURN + FOCUS + A11Y`。

## 5. 验证（Verification）

- `V1` 路径链完整性：路径链结构必须满足 `CHAIN + SEGMENT + CURRENT` 约束；路径片段顺序、当前位置与层级链关系完整，且路径片段映射关系与当前位判定条件可判定。
- `V2` 回溯跳转闭合有效性：回溯跳转闭合结构必须满足 `RETURN + CURRENT` 约束；上游片段跳转、当前位置停驻、回溯出口与回溯后当前位置落位可解释，且回溯关系沿既有路径链成立。
- `V3` 路径链可达一致性：路径朗读顺序、当前位置提示与跳转说明必须满足 `FOCUS + A11Y` 约束；并对辅助技术可感知。
- `V4` 最小必要性：移除任一 `B*` 后，`V1 ~ V3` 至少一项失败。
- `V5` 结论表达：逐条输出 `R* -> C* + 边界绑定` 或 `R* -> N* + 边界绑定` 的通过或失败结论，并附对应边界证据。
