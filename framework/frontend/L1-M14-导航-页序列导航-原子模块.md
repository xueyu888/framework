# 前端页序列导航原子模块:FrontendPageSequenceNavigationAtoms

@framework

模块短标签：页窗导航
典型落地对象：pagination / page window / page jumper
不包括：一般集合浏览、路径链导航、互斥视图切换

本模块定义前端页序列导航原子。它承接分页条、页窗跳转和页码序列导航这类“围绕当前页组织前后推进、跳页关系与页窗落位”的结构，不把无限滚动、层级路径导航或任意自由跳转面混进同一模块。

## 1. 能力声明（Capability Statement）

- `C1` 页序列承载能力：把页号序列、当前页与页窗关系收敛为稳定页序列承载结构。
- `C2` 页跳转闭合能力：把前后页跳转、页码选择、页窗切换出口与跳转后当前位置落位收敛为稳定页跳转闭合结构。
- `C3` 分页可达能力：把当前页提示、键盘移动与分页状态说明收敛为稳定可达结构。

### 非职责声明（Non-Responsibility Statement）

- `N1` 非职责声明：不负责列表内容本体、无限滚动承接与页面级路由组织。

## 2. 边界定义（Boundary / 参数）

- `PAGE` 页边界：页号序列、当前页、页窗范围与当前页到页窗的映射关系需稳定。来源：`C1 + C3`。
- `WINDOW` 页窗边界：可见页窗、折叠页段、页窗切换关系与页窗展开条件需稳定。来源：`C1 + C2`。
- `JUMP` 跳转边界：上一页、下一页、页码跳转、再次跳转出口与跳转结果类型需稳定。来源：`C2 + C3`。
- `STATE` 状态边界：可跳转、不可跳转与加载中分页等状态集合、状态迁移关系与反馈窗口需稳定。来源：`C2 + C3`。
- `FOCUS` 焦点边界：页码聚焦、页窗移动、回退落点与焦点回路需稳定。来源：`C1 + C3`。
- `A11Y` 可访问边界：当前页提示、页窗说明、跳转说明与辅助技术可感知关系需可达。来源：`C1 + C3`。

## 3. 最小可行基（Minimum Viable Bases）

- `B1` 页序列承载基：L0.M0[R1,R3] + L0.M1[R3] + L0.M2[R1,R3]。来源：`C1 + PAGE + WINDOW + FOCUS`。
- `B2` 页跳转闭合基：L0.M0[R4] + L0.M1[R4] + L0.M2[R1,R3]。来源：`C2 + C3 + JUMP + STATE + A11Y`。

## 4. 基组合原则（Base Combination Principles）

- `R1` 页序列承载先行
  - `R1.1` 参与基：`B1`。
  - `R1.2` 组合方式：先固定页号序列、当前页、页窗范围与当前页到页窗的映射关系，再允许页跳转出口接入。
  - `R1.3` 输出能力：`C1`。
  - `R1.4` 边界绑定：`PAGE + WINDOW + FOCUS`。
- `R2` 页跳转闭合成立
  - `R2.1` 参与基：`B1 + B2`。
  - `R2.2` 组合方式：当前页必须能在页序列中被稳定判定，并通过上一页、下一页、页码跳转、页窗切换、跳转结果类型与跳转后当前位置落位形成统一分页导航结构；若不能围绕页窗口形成前后推进、跳页与当前位置重落位关系，或退化为一般集合浏览、路径链导航或任意自由跳转面，则该结构不成立。
  - `R2.3` 输出能力：`C1 + C2 + C3`。
  - `R2.4` 边界绑定：`JUMP + STATE + A11Y + PAGE + WINDOW + FOCUS`。
- `R3` 禁止组合
  - `R3.1` 参与基：`B1 + B2`。
  - `R3.2` 组合方式：没有当前页却声明页跳转、页窗关系不可解释、跳转说明缺位，或把一般集合浏览、路径链导航、互斥视图切换误写为页序列导航原子的方案均无效。
  - `R3.3` 失效结论：`N1`。
  - `R3.4` 边界绑定：`PAGE + WINDOW + JUMP + STATE + FOCUS + A11Y`。

## 5. 验证（Verification）

- `V1` 页序列承载完整性：页序列结构必须满足 `PAGE + WINDOW` 约束；页号序列、当前页与页窗范围完整，且当前页到页窗的映射关系与页窗展开条件可判定。
- `V2` 页跳转闭合有效性：页跳转结构必须满足 `JUMP + STATE` 约束；前后页、页码选择、页窗切换与跳转后当前位置落位可解释，且跳转结果类型完整。
- `V3` 分页可达一致性：当前页提示、跳转说明与焦点移动必须满足 `FOCUS + A11Y` 约束；并对辅助技术可感知。
- `V4` 最小必要性：移除任一 `B*` 后，`V1 ~ V3` 至少一项失败。
- `V5` 结论表达：逐条输出 `R* -> C* + 边界绑定` 或 `R* -> N* + 边界绑定` 的通过或失败结论，并附对应边界证据。
