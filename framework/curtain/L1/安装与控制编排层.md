# 窗帘框架模块标准（L1）安装与控制编排层
<!--@layer module=curtain; level=L1-->

## 1. 目标（Goal）

将底层部件能力编排为可安装、可控制、可验证的中间能力层。

## 2. 边界定义（Boundary）

- `INSTALL_SCOPE`：安装定位与固定边界。
- `CONTROL_SCOPE`：开合控制与停靠策略边界。
- `SAFETY_SCOPE`：故障保护与安全动作边界。

## 3. 最小可行基（Bases）

- `B1` 安装定位基  <!--@base id=B1; name=INSTALL_BASE; capability=installation_alignment-->
- `B2` 控制编排基  <!--@base id=B2; name=CONTROL_BASE; capability=control_orchestration-->
- `B3` 安全治理基  <!--@base id=B3; name=SAFETY_BASE; capability=safety_governance-->

## 4. 组合原则（Combination Principles）

仅允许下游向上游组合：`L1 -> L2`。禁止同层组合与反向组合。

组合约束集合：
- `C1` 相邻层约束
- `C2` 下游上行约束
- `C3` 能力契约约束

组合映射（示例）：
- `U1 = {L1.B1, L1.B2} --[C1,C2,C3]--> L2.B1`

- 安装定位基组合到幕布组件。  <!--@compose from=L1.Mcurtain.B1; to=L2.Mcurtain.B1; rule=adjacent_only; principle=P_DOWNSTREAM_UPSTREAM; constraint=C1_C2_C3-->
- 控制编排基组合到导轨组件。  <!--@compose from=L1.Mcurtain.B2; to=L2.Mcurtain.B2; rule=adjacent_only; principle=P_DOWNSTREAM_UPSTREAM; constraint=C1_C2_C3-->
- 安全治理基组合到驱动控制组件。  <!--@compose from=L1.Mcurtain.B3; to=L2.Mcurtain.B3; rule=adjacent_only; principle=P_DOWNSTREAM_UPSTREAM; constraint=C1_C2_C3-->

## 5. 验证（Verification）

- 每个 L1 基必须至少指向一个 L2 父节点。
- 每个 L2 基必须至少由一个 L1 子节点支撑。
- 不允许同层或跨层跳级组合。
