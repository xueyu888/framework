# Shelf AI 0.0.39

## 中文说明

- 这一版把 Shelf AI 从“仅做校验提示的伴随插件”推进成“后台工作区守卫”。相关文件一旦保存、重命名、删除、被外部修改，或者窗口重新聚焦，扩展就会先判断这次变更是否需要自动物化、是否需要补跑 `mypy`、以及是否命中了 `projects/*/generated/*` 禁区。
- 新增 `normal` / `strict` 两档守卫模式。`normal` 模式会把 generated 直改明确报错；`strict` 模式会尝试通过项目物化把 generated 产物恢复回正确状态，并把恢复失败当成硬问题返回。
- 这一版还补上了 git hooks 状态检测与一键安装入口，并把相关配置、README、回归测试以及中文架构演示文稿的生成脚本一并同步进仓库，方便团队推广与讲解。

## English Notes

- This release moves Shelf AI from a validation-only companion toward a background workspace guard. On relevant saves, file operations, external file changes, and focus returns, the extension now classifies the change first so it can decide whether to auto-materialize, whether to run `mypy`, and whether the change touched the protected `projects/*/generated/*` area.
- It adds `normal` and `strict` guard modes. Normal mode reports direct generated-artifact edits, while strict mode attempts to restore those artifacts through project materialization and treats restore failure as a hard guard issue.
- The release also adds git-hook status detection with a one-click install path, refreshes the packaged configuration and README, expands regression coverage, and includes a repository-side presentation generator that explains the current Shelf AI structure and Codex-facing guard model.

## 安装 / Install

- 从本页面下载对应版本的 VSIX 安装包
- 在 VS Code 中执行 `Extensions: Install from VSIX...`
- 如果你使用 WSL，请在对应的 WSL 工作区窗口里安装并重载当前窗口
