# Shelf AI 0.0.38

## 中文说明

- 这一版同步了 Shelf AI 的打包元数据，使扩展里的仓库、问题反馈和主页链接全部指向已经正式更名后的 `xueyu888/shelf`。
- 同时整理了仓库首页文案，进一步强调 Shelf 的核心不是“凭感觉设计”，而是先把设计写成逻辑自洽、边界清晰、可验证的结构语言，再让 AI 落代码。
- 这不是一版功能侧更新，主要是面向发布入口、品牌一致性和公开定位的整理版。

## English Notes

- This release syncs Shelf AI packaged metadata to the renamed `xueyu888/shelf` repository, so repository, issue-tracker, and homepage links now resolve to the live public repo.
- It also refreshes the repository-facing positioning so Shelf emphasizes logically self-consistent design, not intuition-led structure, before AI implementation begins.
- This is not a feature release. It is a packaging, branding, and public-entry consistency update.

## 安装 / Install

- 从本页面下载对应版本的 VSIX 安装包
- 在 VS Code 中执行 `Extensions: Install from VSIX...`
- 如果你使用 WSL，请在对应的 WSL 工作区窗口里安装并重载当前窗口
