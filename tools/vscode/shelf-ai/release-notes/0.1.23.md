# 0.1.23

## 中文说明

本说明覆盖提交区间：`shelf-ai-v0.1.22..shelf-ai-v0.1.23`（包含该区间全部提交）。

- framework 作者语法中的非职责声明进一步收平，`N*` 条目默认按扁平一等作者项书写，不再鼓励嵌套式表达。
- 本版本不引入新的运行时能力，主要是作者侧文档结构与书写风格的收口发布。

对使用者可见的影响：

- framework 文档的非职责声明写法更统一，作者在书写 `N*` 条目时更直接，也更容易与 lint / completion 预期保持一致。
- 本版本属于作者体验收口补丁，不改变 Shelf AI 的主要运行行为。

安装方式：

1. 下载发布资产 `shelf-ai-0.1.23.vsix`
2. 在 VS Code 中执行 `Extensions: Install from VSIX...`
3. 选择该文件并安装

## English Notes

This note covers the full commit range: `shelf-ai-v0.1.22..shelf-ai-v0.1.23`.

- Flattened framework non-responsibility authoring so `N*` items are now authored as first-class flat entries instead of nested-style patterns.
- This release does not add new runtime capabilities; it is a focused authoring-structure cleanup release.

User-visible impact:

- Framework non-responsibility sections are now more uniform and easier to keep aligned with lint and completion expectations.
- This is an authoring-experience cleanup patch rather than a behavioral runtime change.

How to install:

1. Download `shelf-ai-0.1.23.vsix` from release assets
2. Run `Extensions: Install from VSIX...` in VS Code
3. Select the VSIX file and install
