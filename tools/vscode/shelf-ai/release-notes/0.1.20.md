# 0.1.20

## 中文说明

- 本次版本聚焦 Shelf AI 报错文案本地化：Problems 面板中的插件诊断现在默认以中文展示，不再把 generated 保护提示和 correspondence 汇总直接以英文暴露给作者。
- 补齐了 correspondence 校验原因的中文映射，常见的参数边界漂移、规则边界未生效等提示会直接以中文落到默认诊断链路。
- 本地已重新完成插件测试、VSIX 打包与安装验证，确保公开发布资产与当前修复状态一致。

对使用者可见的影响：

- 无论你的 VS Code UI 是中文还是英文，Shelf AI 默认问题提示都会优先显示中文。
- 本次为诊断体验修复版，不引入新的命令面或工作流语义变化。

安装方式：

1. 下载发布资产 `shelf-ai-0.1.20.vsix`
2. 在 VS Code 中执行 `Extensions: Install from VSIX...`
3. 选择该文件并安装

## English Notes

- This release focuses on diagnostics localization: Shelf AI Problems messages now default to Chinese instead of exposing English generated-artifact guard text and correspondence summary reasons.
- Added correspondence-side reason localization so common audit-drift diagnostics, such as boundary-read mismatches, land in Chinese by default.
- The local VSIX was repackaged and reinstalled after re-running plugin validation so the published asset matches the current fix.

User-visible impact:

- Shelf AI diagnostics now default to Chinese regardless of whether your VS Code UI is English or Chinese.
- This is a diagnostics-experience patch release and does not introduce new commands or workflow semantics.

How to install:

1. Download `shelf-ai-0.1.20.vsix` from release assets
2. Run `Extensions: Install from VSIX...` in VS Code
3. Select the VSIX file and install
