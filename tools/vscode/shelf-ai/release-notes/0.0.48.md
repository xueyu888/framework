# 0.0.48

## 中文说明

- Shelf AI 已跟随仓库完成重构，当前主链明确为：
  `Framework -> Config -> Code -> Evidence`。
- 插件里的项目配置跳转已切到 `projects/*/project.toml`，不再把旧双轨配置文件当作主入口。
- 插件侧的自动物化、生成物保护、预检与提示说明都已切到 `project.toml + canonical.json`。
- `projects/*/generated/canonical.json` 是唯一机器真相源；证据树、证据清单和校验报告都按 canonical 派生视图处理。

## English Notes

- Shelf AI now follows the rewritten repository architecture:
  `Framework -> Config -> Code -> Evidence`.
- Extension-side project navigation now targets `projects/*/project.toml`; the removed dual-track config files are no longer treated as live entrypoints.
- Auto-materialization, generated-artifact guarding, codegen preflight, and packaged guidance now all align with `project.toml + canonical.json`.
- `projects/*/generated/canonical.json` is the only machine truth; evidence manifests, evidence trees, and reports are treated as canonical-derived views.
