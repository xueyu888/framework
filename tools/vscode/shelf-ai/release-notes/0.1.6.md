# 0.1.6

## 中文说明

- 本次发布继续收紧 Shelf AI 与四层 canonical 主链的一致性：framework 边界跳转到 `project.toml` 现在只依赖 canonical 中的模块导出投影，不再保留扩展侧的推断式 fallback。
- canonical 中的 `Framework / Config / Code / Evidence` 模块身份进一步稳定化，补齐跨层 `class_id` 与 `source_ref`，避免导航、校验与追溯继续依赖临时运行时对象特征。
- `base_bindings` 已继续细化到 code owner、implementing slot 与 bound symbol 级别，并补充了对应回归测试，确保 Base 追溯不会长期停在 module 归属粒度。

对使用者可见的影响：

- 当 canonical 存在时，Framework 边界导航会更稳定地跳到统一 `project.toml` 配置位置
- 当 canonical 缺失时，扩展不会再偷偷猜测配置 section，而是只保留 framework 本地导航
- 证据树与相关追溯材料会显示更稳定的层内身份与更细的 Base 绑定信息

安装或获取方式：

1. 下载发布资产 `shelf-ai-0.1.6.vsix`
2. 在 VSCode 中执行 `Extensions: Install from VSIX...`
3. 选择该文件完成安装

注意事项：

- `projects/knowledge_base_basic` 仍只用于本地验证、演示和回归测试，不属于公开发布物
- 本次版本包含重新打包并本地安装验证的 VSIX，确保安装产物与当前验证通过的仓库状态一致

## English Notes

- This release tightens Shelf AI against the four-layer canonical mainline: framework boundary jumps into `project.toml` now rely only on canonical-backed module export projections instead of extension-side inferred fallback tables.
- The canonical output now carries more stable `class_id` and `source_ref` identities across `Framework / Config / Code / Evidence`, so navigation, validation, and tracing do not drift back toward temporary runtime object traits.
- `base_bindings` are now traced down to code owner, implementing slot, and bound symbol granularity, with regression coverage to keep Base tracing from stopping at module ownership.

User-visible impact:

- When canonical exists, framework boundary navigation resolves the unified `project.toml` target more reliably
- When canonical is missing, the extension no longer guesses config sections and falls back to framework-local navigation only
- Evidence and tracing views surface more stable layer identities and finer-grained Base binding details

Install / access:

1. Download the release asset `shelf-ai-0.1.6.vsix`
2. Run `Extensions: Install from VSIX...` in VSCode
3. Select the file to complete installation

Notes:

- `projects/knowledge_base_basic` remains a local validation, demo, and regression sample only; it is not a public release deliverable
- This version includes a repackaged and locally reinstalled VSIX so the installable asset matches the validated repository state
