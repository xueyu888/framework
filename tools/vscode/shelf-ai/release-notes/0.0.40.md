# Shelf AI 0.0.40

## 中文说明

- 这一版把 Shelf AI 从“只盯 framework tree”推进到“直接围绕 workspace governance tree 工作”。扩展现在会读取工作区治理树，直接打开治理树视图，并在侧边栏展示最近一次 touched / affected 节点闭包，而不是只展示文件级变化。
- 严格校验也已经和这棵治理树对齐。插件触发的校验与仓库命令行校验现在共享同一个 workspace-governance 入口，相关项目和受影响节点会按树闭包解析，不再只靠 framework-tree 路径启发式。
- 这一版还补了治理树相关回归测试，并更新了打包 README / Changelog，使已发布扩展的说明与当前仓库里的 tree-first 工作流一致。

## English Notes

- This release moves Shelf AI from a framework-tree-only view toward a workspace governance-tree workflow. The extension now reads the workspace governance tree directly, opens that governance-tree view, and shows the recent touched / affected node closure in the sidebar instead of reporting file changes only.
- Strict validation is now aligned with the same governance-tree source. The extension-side guard and the repository CLI validation path resolve affected projects and nodes through workspace-governance closure instead of relying on framework-tree-only path heuristics.
- The release also adds governance-tree regression coverage and refreshes the packaged README / changelog so the published extension matches the repository's current tree-first workflow.

## 安装 / Install

- 从本页面下载对应版本的 VSIX 安装包
- 在 VS Code 中执行 `Extensions: Install from VSIX...`
- 如果你使用 WSL，请在对应的 WSL 工作区窗口里安装并重载当前窗口
