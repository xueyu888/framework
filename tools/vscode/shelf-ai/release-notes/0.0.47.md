## 中文说明

- 这一版把 Shelf AI 的作者工作流从“单文件假设”升级成了结构化入口。插件现在支持 `framework_drafts/` 草稿区导航和一键发布到正式 `framework/`，也支持直接从插件脚手架生成一个框架驱动项目，不再要求作者先手工拼好 `product_spec.toml` 和目录结构。
- `product_spec.toml` 现在正式支持“根入口 + section 分文件”模式：`project` 与 `framework` 仍保留在根文件中，具体产品真相可以拆到 `product_spec/*.toml`。边界跳转、治理追踪、生成证据和严格校验都会回到真实 section 文件，而不是只能落到根文件。
- Shelf AI 新增了更可控的工作流开关。校验触发模式现在可以选 `manual / save / all`，避免编辑时被过度打断；同时新增 `Shelf: Run Codegen Preflight`，用于在让 AI 生成代码前，先统一跑物化、框架引用关系和严格校验。
- 这一版还正式收口了框架协议里的正负语义：`C*` 只表示正向能力，`N*` 表示非职责声明；禁止/无效组合改为使用 `失效结论`，不再把负向项伪装成 `输出能力`。仓库里的框架文档、模板片段、IR 解析器、严格校验和回归测试都已同步迁移。
- 安装方式：下载本版本附带的 `shelf-ai-0.0.47.vsix`，在 VS Code 中执行 `Extensions: Install from VSIX...` 安装。若升级前工作区或树图页已经打开，安装后建议执行一次 `Developer: Reload Window`，确保新的作者工作流命令、触发模式和导航逻辑完全生效。

## English Notes

- This release turns Shelf AI's authoring flow into a structural workflow instead of a single-file assumption. The extension now supports `framework_drafts/` navigation plus one-click publish into the formal `framework/` tree, and it can scaffold a framework-driven project directly from the extension instead of requiring manual `product_spec.toml` bootstrapping first.
- `product_spec.toml` now officially supports a "root entry + split section files" model: `project` and `framework` stay in the root file, while concrete product-truth sections can live under `product_spec/*.toml`. Boundary navigation, governance tracing, generated evidence, and strict validation all resolve back to the real section file instead of only the root file.
- Shelf AI also adds more controllable workflow gates. Validation trigger mode can now be `manual`, `save`, or `all`, reducing noisy interruptions while editing, and a new `Shelf: Run Codegen Preflight` command lets you run materialization, framework-reference checks, and strict validation before asking AI to generate code.
- This release also formalizes positive vs. negative framework semantics. `C*` is now positive-only, `N*` carries non-responsibility declarations, and prohibited / invalid combinations use `失效结论` instead of pretending that negative statements are `输出能力`. The framework corpus, snippet templates, IR parser, strict validator, and regression tests were migrated together.
- Install by downloading the bundled `shelf-ai-0.0.47.vsix` and running `Extensions: Install from VSIX...` in VS Code. If the workspace or tree pages were already open before upgrading, run `Developer: Reload Window` after installation so the new authoring commands, trigger modes, and navigation behavior are fully active.
