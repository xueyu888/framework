## 中文说明

- 本次发布更新了 Shelf AI 的插件视觉识别：README / Marketplace 入口现在使用新的彩色小黄鸭图标，VS Code Activity Bar 则改为专用的单色 duck glyph，以适配侧边栏图标的单色显示约束。
- 仓库中补充保留了本次图标迭代的设计稿与预览页，后续若需要继续微调插件图标或 README 展示图，可直接沿现有设计资产继续收敛。
- 已完成本地插件测试、仓库校验、VSIX 打包与安装验证，发布资产包含 `shelf-ai-0.1.11.vsix`。

对使用者可见的影响：

- 插件命令、校验逻辑与主链行为没有变化；这次是视觉识别与发布资产更新。

安装方式：

1. 下载发布资产 `shelf-ai-0.1.11.vsix`
2. 在 VS Code 中执行 `Extensions: Install from VSIX...`
3. 选择该文件并安装

注意事项：

- Activity Bar 图标遵循 VS Code 单色 SVG 约束，因此它会比 README / Marketplace 主图更简化，这是有意的专用适配，而不是资源丢失。

## English Notes

- This release refreshes Shelf AI branding: the README / Marketplace entry now uses a new colorful duck icon, while the VS Code Activity Bar uses a dedicated monochrome duck glyph tuned for sidebar icon constraints.
- The repository now keeps the icon exploration assets and preview files, so future plugin-icon or README-image tweaks can continue from the current design source instead of starting over.
- Local plugin tests, repository validation, VSIX packaging, and install verification were rerun. The release asset includes `shelf-ai-0.1.11.vsix`.

User-visible impact:

- Plugin commands, validation behavior, and mainline architecture are unchanged; this is a branding / release-asset refresh.
- After install or upgrade, the extension list, README hero image, and Activity Bar icon will all reflect the new duck-based icon system.

How to install:

1. Download `shelf-ai-0.1.11.vsix` from the release assets
2. Run `Extensions: Install from VSIX...` in VS Code
3. Select the VSIX file and install

Notes:

- The Activity Bar icon follows VS Code's monochrome SVG constraint, so it is intentionally simpler than the README / Marketplace artwork rather than a missing-asset regression.
