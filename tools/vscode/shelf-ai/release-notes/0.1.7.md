# 0.1.7

## 中文说明

- 本次发布把 Shelf AI 的树图能力正式重构为 runtime webview 子系统，明确拆分为 `tree_runtime_models / tree_webview_bridge / webview/tree(model-layout-render-bridge)`，不再依赖旧的单体式 extension 内嵌渲染实现。
- 框架树视觉与交互已恢复到更强层次感的圆卡节点方案：保留分栏层带、旧版风格箭头与关系线、节点 hover 元数据、节点/边选中、高亮以及默认收起的右侧 inspector。
- 继续推进 settings-first 策略：框架树横向/纵向间距、tree zoom 上下限、滚轮灵敏度、inspector 尺寸，以及 validation timeout / debounce / generated suppression / stale restart threshold 现在都可以直接在 VSCode 插件设置中调整。

对使用者可见的影响：

- 框架树默认观感更接近历史高可读版本，同时保持当前 canonical-first runtime 架构
- 已打开的 tree panel 在相关 tree setting 变化时可自动重渲染，不必每次都改代码常量
- validation 相关时间阈值可以按机器性能和仓库规模自行调节

安装或获取方式：

1. 下载发布资产 `shelf-ai-0.1.7.vsix`
2. 在 VSCode 中执行 `Extensions: Install from VSIX...`
3. 选择该文件完成安装

注意事项：

- 这次发布不会放宽 canonical / freshness / generated guard 的正式约束；新增 setting 只作用于本地工作台体验或运行时调优
- 若你刚从本地旧版切换到本版本，首次安装后仍建议执行一次 `Developer: Reload Window`

## English Notes

- This release turns the Shelf AI tree experience into a formal runtime webview subsystem with explicit `tree_runtime_models / tree_webview_bridge / webview/tree(model-layout-render-bridge)` layering instead of the previous monolithic extension-side renderer.
- The framework tree now returns to a denser round-node visual baseline with grouped depth bands, old-style arrows and edge lines, richer hover metadata, node/edge inspection, highlighting, and a default-collapsed inspector.
- The extension continues its settings-first direction: framework tree spacing, tree zoom bounds, wheel sensitivity, inspector sizing, and validation timeout / debounce / generated suppression / stale restart threshold are now adjustable directly from VSCode settings.

User-visible impact:

- The framework tree is closer to the earlier high-readability visual baseline while keeping the current canonical-first runtime architecture
- An open tree panel can re-render automatically when related tree settings change, so tuning no longer requires editing code constants
- Validation timing thresholds can now be tuned to match machine performance and repository size

Install / access:

1. Download the release asset `shelf-ai-0.1.7.vsix`
2. Run `Extensions: Install from VSIX...` in VSCode
3. Select the file to complete installation

Notes:

- This release does not relax canonical, freshness, or generated-guard constraints; the new settings only affect local workbench behavior or runtime tuning
- If you are upgrading from an older local install, a one-time `Developer: Reload Window` is still recommended after installation
