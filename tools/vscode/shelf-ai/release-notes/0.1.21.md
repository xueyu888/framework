# 0.1.21

## 中文说明

本说明覆盖提交区间：`shelf-ai-v0.1.20..shelf-ai-v0.1.21`（包含该区间全部提交）。

- 新增 framework 文档实时语法诊断（on-type lint）与 Quick Fix：在编辑时即可看到 Problems 波浪线与修复建议，不再依赖保存后才发现格式问题。
- 门禁语义收敛到“授权阶段（grant-stage）”：意图门禁聚焦在“需求到 framework 映射授权”阶段，保存阶段不再作为默认阻断点；并同步了相关配置与测试。
- framework 作者入口能力增强：补全与片段支持上下文编号推断；在“边界定义”章节输入 `-` 可直接自动展开为参数条目模板并推断下一个 `P` 编号。
- 章节命名统一与全链路对齐：统一使用 `## 2. 边界定义（Boundary / Parameter 参数）`、`## 3. 最小结构基（Minimal Structural Bases）`，并同步 parser / lint / completion / navigation / snippets / tests / framework 文档。
- 完成发布前重建与校验：已重建 webview bundle、通过插件测试、硬编码门禁、mypy 与 `validate_canonical --check-changes`，并重新打包安装本地 VSIX。

对使用者可见的影响：

- framework Markdown 在编辑阶段会得到更接近代码语言的即时错误提示与修复建议。
- 默认作者语法以“边界定义 / 最小结构基”为准，且边界章节条目录入速度更快（`-` 触发模板）。
- 本版本包含作者体验与门禁行为收敛，不是单一文案修复。

安装方式：

1. 下载发布资产 `shelf-ai-0.1.21.vsix`
2. 在 VS Code 中执行 `Extensions: Install from VSIX...`
3. 选择该文件并安装

## English Notes

This note covers the full commit range: `shelf-ai-v0.1.20..shelf-ai-v0.1.21`.

- Added real-time framework Markdown diagnostics (on-type lint) plus Quick Fix actions, so format issues are surfaced while editing instead of only after save.
- Refined intent-gate behavior to grant-stage authorization semantics: the gate is enforced at mapping/grant time rather than as a default save-time blocker, with related config/test alignment.
- Improved framework authoring ergonomics: completion/snippets now infer numbering from context, and typing `-` in the Boundary section auto-expands into a parameter entry template with inferred next `P` index.
- Unified section naming across the full chain to `## 2. 边界定义（Boundary / Parameter 参数）` and `## 3. 最小结构基（Minimal Structural Bases）`, including parser/lint/completion/navigation/snippets/tests/framework docs.
- Rebuilt and revalidated release artifacts before packaging: webview build, extension tests, hardcode guard, mypy, and `validate_canonical --check-changes` all passed.

User-visible impact:

- Framework Markdown authoring now has code-like immediate diagnostics with actionable fixes.
- Default authoring terminology is now aligned to “Boundary definition / Minimal structural bases”, and boundary item entry is faster via dash-triggered expansion.
- This release is a combined authoring-experience and gate-behavior alignment, not a single-session patch.

How to install:

1. Download `shelf-ai-0.1.21.vsix` from release assets
2. Run `Extensions: Install from VSIX...` in VS Code
3. Select the VSIX file and install
