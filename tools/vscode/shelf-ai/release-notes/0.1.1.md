# 0.1.1

## 中文说明

- 修复了 Shelf 默认校验命令仍携带过期参数的问题，避免扩展在触发 canonical 校验时直接报错。
- 增加了运行时命令归一化逻辑；即使用户本地还保留旧的 `--json` 配置，Shelf 也会自动剥离该参数后再执行。
- 同步更新 README 与回归测试，使扩展文档、默认配置和仓库脚本契约保持一致。

## English Notes

- Fixed Shelf validation defaults so they no longer pass stale arguments to `validate_canonical.py`.
- Added runtime command normalization so existing user settings with the stale `--json` flag still execute successfully.
- Updated the README and regression tests so extension docs, default settings, and repository command contracts stay aligned.
