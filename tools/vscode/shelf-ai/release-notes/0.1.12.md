## 中文说明

- 本次发布只微调 Shelf AI 的 Activity Bar 专用小黄鸭图标：主体占比继续收紧，线条也进一步变细，让侧边栏里的识别度和精致度更平衡。
- README / Marketplace 使用的彩色主图没有变化；这次只针对 VS Code 侧边栏单色 SVG 版本做适配修正。
- 已完成本地插件测试、仓库校验、VSIX 打包与安装验证，发布资产包含 `shelf-ai-0.1.12.vsix`。

对使用者可见的影响：

- 插件命令、校验逻辑与仓库主链行为没有变化。
- 安装或升级后，Activity Bar 中的鸭子图标会比 `0.1.11` 更贴边、线条更轻一些。

安装方式：

1. 下载发布资产 `shelf-ai-0.1.12.vsix`
2. 在 VS Code 中执行 `Extensions: Install from VSIX...`
3. 选择该文件并安装

注意事项：

- 这是一次图标精修补丁发布，不引入新的插件功能或配置项。
