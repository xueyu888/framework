# 0.1.0

## 中文说明

- 这次版本号提升到 `0.1.0`，因为 Shelf AI 对齐的是一次仓库级架构重写，而不是普通的 `0.0.x` 小修补。
- 插件继续围绕新的主链工作：
  `Framework Markdown -> Package Registry -> Project Config -> Code -> Evidence`。
- 项目配置跳转、自动物化、生成物保护、预检和树视图都继续以 `projects/*/project.toml` 与 `generated/canonical_graph.json` 为基础。
- 仓库文档现在明确区分“已经落地的主骨架”和“尚未完全切净的场景级聚合残留”，不再把当前状态表述得比实际更满。

## English Notes

- The version is now `0.1.0` because Shelf AI is aligned with a repository-wide architecture rewrite rather than another `0.0.x` patch.
- The extension continues to follow the rewritten mainline:
  `Framework Markdown -> Package Registry -> Project Config -> Code -> Evidence`.
- Project-config navigation, auto-materialization, generated-artifact guarding, codegen preflight, and tree views remain grounded in `projects/*/project.toml` and `generated/canonical_graph.json`.
- Repository documentation now distinguishes between the live rewritten main skeleton and the remaining scene-level aggregate runtime pieces instead of overstating the rewrite as fully complete.
