# 程序预标注:ProgrammaticPreAnnotation

@framework

## 0. 模块身份（Module Identity）

    模块名 := 程序预标注:ProgrammaticPreAnnotation
    问题域 := 预标注流程编排域
    目标   := 程序预标注模块负责初步对文本进行标签标注，判定在某文本中，某标签是否被启用或者需要交由人工进行审核，采用调用大模型的方式来实现。为了保证结果可控，大模型只负责对文本和标签的匹配程度进行评分。后续再根据评分判定是否启用。评分后的处理过程中间产物需要输出，用于结果追溯。大模型评分和评分判定之间需要解耦。评分结果需要实现可解释性，可控性。以及满足后续的审核需求。
    模块型 := 编排型

## 1. Base

    B1 大模型服务 := L0.M0，负责对输入的文本和标签进行大模型评分，得到大模型原始响应。
    B2 原始响应解析器 := L0.M1，负责将原始响应解析为 raw_score 和 reason。
    B3 线性映射归一化器 := L0.M2，负责将 raw_score 映射为 normalized_score。
    B4 标签启用自动标注器 := L0.M3，负责根据 normalized_score 导出满足 L-1.M4 的自动标注结果。

## 2. Combination Principles

只写原则，不写具体结果类。
记号：`x ≡ y` 表示同一结果类；`nf(x)` 表示 `x` 的规范形。

    R1 评分结果解析衔接原则:
      formation     := combinable(B1, B2) iff B1 导出的原始响应可被 B2 消费。
      satisfaction  := satisfied(B1, B2) iff B2 对 B1 的输出完成解析，并导出 raw_score 和 reason。
      identity      := <B1, B2> ≡ <B1', B2'> iff 两者在原始响应到解析结果的衔接语义上等价。
      normalization := nf(<B1, B2>) = <评分调用, 结果解析> iff 该组合满足原始响应解析闭环。

    R2 评分归一化衔接原则:
      formation     := combinable(B2, B3) iff B2 导出的 raw_score 可被 B3 消费。
      satisfaction  := satisfied(B2, B3) iff B3 基于 B2 的 raw_score 导出 normalized_score。
      identity      := <B2, B3> ≡ <B2', B3'> iff 两者在解析结果到归一化结果的衔接语义上等价。
      normalization := nf(<B2, B3>) = <结果解析, 评分归一化> iff 该组合满足评分归一化闭环。

    R3 自动标注判定衔接原则:
      formation     := combinable(B3, B4) iff B3 导出的 normalized_score 可被 B4 消费。
      satisfaction  := satisfied(B3, B4) iff B4 基于 B3 的 normalized_score 导出自动标注结果。
      identity      := <B3, B4> ≡ <B3', B4'> iff 两者在归一化结果到自动标注结果的衔接语义上等价。
      normalization := nf(<B3, B4>) = <评分归一化, 自动标注判定> iff 该组合满足自动标注判定闭环。

## 3. Result Space

    A1 解析型预标注片段:
      form    := <B1, B2>
      by      := R1
      meaning := 从评分调用到解析结果的局部编排结果，表达原始响应已被恢复为 raw_score 和 reason。

    A2 归一化型预标注片段:
      form    := <B1, B2, B3>
      by      := R1, R2
      meaning := 从评分调用到归一化结果的局部编排结果，表达流程已形成 normalized_score。

    A3 完整程序预标注流程:
      form    := <B1, B2, B3, B4>
      by      := R1, R2, R3
      meaning := 程序预标注的完整编排结果；接口内容必须满足 L-1.M5。
