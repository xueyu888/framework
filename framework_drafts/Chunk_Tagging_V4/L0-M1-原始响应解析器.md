# 原始响应解析器:RawResponseParser

@framework

## 0. 模块身份（Module Identity）

    模块名 := 原始响应解析器:RawResponseParser
    问题域 := 响应解析处理域
    目标   := 原始响应解析器负责接收大模型原始响应，并依据语义映射关系提取原始评分结果与评分原因。该模块只声明原始响应解析所需的内部构件、输入输出接口和解析约束，不负责发起大模型调用、文本标签身份装配、归一化计算或阈值判定。
    模块型 := 处理型

## 1. Base

    B1 原始响应到目标输出的映射关系 := 原始响应中的内容和目标输出接口协议要求的内容之间的映射关系。原始响应.raw_score -> 目标输出.raw_score；原始响应.reason -> 目标输出.reason。

## 2. Combination Principles

只写原则，不写具体结果类。
记号：`x ≡ y` 表示同一结果类；`nf(x)` 表示 `x` 的规范形。

    R1 原始响应解析规则:
      formation     := combinable(B1) iff B1 接收单个原始响应输入，且原始响应中的内容可唯一映射到目标输出字段。
      satisfaction  := satisfied(B1) iff B1 从原始响应中解析出 raw_score 和 reason。
      identity      := <B1> ≡ <B1'> iff 两者在原始响应到解析结果的字段映射语义上等价。
      normalization := nf(<B1>) = <原始响应解析> iff 该组合满足原始响应到结构化解析结果的闭环。

## 3. Result Space

    A1 解析结果:
      form    := PARSED_RESULT_OUT
      by      := R1
      meaning := 由原始响应解析得到的结果，包括 raw_score 和 reason。
