# 线性映射归一化器:LinearNormalizer

@framework

## 0. 模块身份（Module Identity）

    模块名 := 线性映射归一化器:LinearNormalizer
    问题域 := 评分归一化处理域
    目标   := 线性映射归一化器负责接收一个输入值，并按照线性映射公式将该值归一化到 0~1 范围内。该模块只声明线性映射规则、归一化区间参数与归一化结果导出，不负责输入值来源追踪或非线性归一化。
    模块型 := 处理型

## 1. Base

    B1 线性映射规则 := 负责依据公式 y = (x - a) / (b - a) 执行归一化计算，其中 x 为原始评分，a 为归一化下界，b 为归一化上界。

## 2. Combination Principles

只写原则，不写具体结果类。
记号：`x ≡ y` 表示同一结果类；`nf(x)` 表示 `x` 的规范形。

    R1 线性映射归一化规则:
      formation     := combinable(B1) iff B1 接收单个数值输入，且归一化区间参数满足 P1, P2, P3, P4, P5, P6, P7。
      satisfaction  := satisfied(B1) iff B1 依据线性映射公式对输入值完成归一化，并导出 normalized_score。
      identity      := <B1> ≡ <B1'> iff 两者在线性映射公式、区间约束和归一化输出语义上等价。
      normalization := nf(<B1>) = <线性归一化> iff 该组合满足从原始评分到 0~1 结果的线性映射闭环。

## 3. Result Space

    A1 归一化结果:
      form    := NORMALIZED_SCORE_OUT
      by      := R1
      meaning := 由原始评分映射得到的归一化结果；结果范围为 0~1。

### 4.2 Parameter

      affects := R1, IO

    P2 归一化上界类型:
      domain  := 归一化上界必须为数值类型。
