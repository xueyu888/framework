# 线性映射归一化器:LinearNormalizer

@framework

## 0. 目标 (Goal)
- 线性映射归一化器负责接收一个输入值，并按照线性映射公式将该值归一化到 `0~1` 范围内。
- 该模块只声明线性映射规则、归一化区间参数与归一化结果导出，不负责输入值来源追踪或非线性归一化。

## 1. 最小结构基（Minimal Structural Bases）

- `B1` 线性映射规则：负责依据公式 `y = (x - a) / (b - a)` 执行归一化计算，其中 `x` 为原始评分，`a` 为归一化下界，`b` 为归一化上界。

## 2. 基排列组合（Base Arrangement / Combination）

- `R1` `线性映射归一化规则`：由 `{B1}` 消费`VALUE_IN`形成 `NORMALIZED_SCORE_OUT`，导出 `C1 + C2`。

## 3. 边界定义（Boundary）

### 3.1 接口定义（IO / Ports）

- `VALUE_IN`：运行时输入接口。载荷类型为数值；基数为 `1`；语义为待归一化的单个输入值。
- `NORMALIZED_SCORE_OUT`：运行时输出接口。载荷类型为数值；基数为 `1`；语义为由原始评分映射得到的归一化结果；结果范围为 `0~1`。

### 3.2 参数边界（Parameter Constraints）

- `P1` 归一化下界类型：归一化下界必须为数值类型。
- `P2` 归一化上界类型：归一化上界必须为数值类型。
- `P3` 下界来源一致性：归一化下界必须与 `L-1.M1[R1]` 定义的原始评分下界一致。
- `P4` 上界来源一致性：归一化上界必须与 `L-1.M1[R1]` 定义的原始评分上界一致。
- `P5` 区间有效性：归一化下界必须小于归一化上界。
- `P6` 输入评分下界约束：原始评分必须大于等于归一化下界。
- `P7` 输入评分上界约束：原始评分必须小于等于归一化上界。
- `P8` 结果范围约束：归一化结果必须落在 `0~1` 范围内。

## 4. 能力声明（Capability Statement）

- `C1` 线性映射归一化能力：在输入值和归一化区间参数合法时，能够按线性映射公式完成归一化计算。
- `C2` 归一化结果导出能力：在归一化计算完成时，能够稳定导出单个归一化结果。
- `N1` 非职责声明：不负责判断输入值来自哪个上游模块。
- `N2` 非职责声明：不负责执行分段映射、对数映射或其他非线性归一化规则。
- `N3` 非职责声明：不负责归一化结果之后的阈值判定、审核流转或启用决策。
