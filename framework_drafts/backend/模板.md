## 2. Combination Principles

只写原则，不写具体结果类。
记号：`x ≡ y` 表示同一结果类；`nf(x)` 表示 `x` 的规范形。

    R1 <PrincipleName>:
      formation     := combinable(<...>) iff <...>
      satisfaction  := satisfied(<...>) iff <...>
      identity      := <...> ≡ <...> iff <...>
      normalization := nf(<...>) = <...> iff <...>

    R2 <PrincipleName>:
      formation     := ...
      satisfaction  := ...
      identity      := ...
      normalization := ...

    R3 <PrincipleName>:
      ...
