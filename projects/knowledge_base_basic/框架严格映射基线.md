# knowledge_base_basic 框架严格映射基线

## 1. 当前基线

`knowledge_base_basic` 已经切换到新架构基线：

```text
framework/*.md
  -> framework package registry
    -> projects/knowledge_base_basic/project.toml
      -> package compile
        -> runtime assembly
          -> generated/canonical_graph.json
            -> derived governance views
```

这里没有旧双轨配置主路径，也没有旧项目级聚合模块主路径。
