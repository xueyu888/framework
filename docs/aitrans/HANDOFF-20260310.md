### 3.1 Product Spec 当前固定了什么

产品真相已经固定为：

- 入口：托盘 + 全局快捷键
- 截图方式：区域截图为主，也允许全屏截图
- 结果交付：悬浮面板
- 结果动作：复制、重试翻译、关闭、重新截图
- 平台范围：Windows / macOS
- 治理约束：隐私模式、稳定更新通道、兼容矩阵、阶段时延预算

### 3.2 Implementation Config 当前固定了什么

实现细化已经固定为：

- 宿主：`electron`
- 截图链：`desktop_capturer + native_image_png`
- OCR 链：`macos_vision -> tesseract`
- 翻译链：`openai_translation -> fallback_stub`
- 浮层渲染：`floating_panel_html_v1`
- 发布目标：`dmg + nsis`

注意：这些只是当前选定实现路径，不应再回写到框架层。
