# 父任务状态记录表

以父任务 id 作为索引, 建立的 b+ 树结构:

| 父任务 id | 总子任务数 | 已完成子任务数 |  父任务状态   | 父任务关心的其他信息(必须和父任务id一一对应) |
| :-------: | :--------: | :------------: | :-----------: | :------------------------------------------: |
|   t1234   |     5      |       3        | 审核中/已完成 |             如时间, 任务内容...              |
