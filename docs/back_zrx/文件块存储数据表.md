# 文件块存储数据表

以文件块索引号为索引, 建立的 b+ 树结构:

| 文件块索引号 | 对应文件号 | 文件块内容 | 其他信息 |
| :----------: | :--------: | :--------: | :--------: |
|     0b3c     |   8c6fed   | xxxxxxssss | 时间, 置信度... |

由于需要用文件号查到所属文件块, 还需要以文件号+文件块索引号,l 再建立一棵b+树

| 文件号 | 文件块索引号 | 文件块内容 | 其他信息 |
| :----------: | :--------: | :--------: | :--------: |
| 8c6fed | 0b3c | xxxxxxssss | 时间, 置信度... |

在逻辑上只有一张表, 物理上存在两棵树
