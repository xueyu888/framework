# Framework 实例化规范说明

## 文件描述
本目录用于维护项目级框架实例化规范“定实现”。

## 设计目的
1. 将 `/L2` 下的《前端框架标准》作为后续项目规范生成的基础规范，并可结合团队环境与习惯进行实例化。
2. 在具体项目规范中，支持按需求定制所需特性。
3. 支持 AI 基于 `framework` 目录下的具象化项目规范生成目标项目工程。

## 前置说明
本目录下的实例规范均由 `/framework/L0/前端框架标准.md` 生成，无需在使用流程中重复读取。

## 使用方式
1. 选择或新增 `framework` 目录下的项目实例化规范。
2. 基于实例化规范生成或校验项目工程。
